# RotateView
RotateView Library

> Step 1. Add the JitPack repository to your build file

```Setting.gradle
repositories {
			mavenCentral()
			maven { url 'https://jitpack.io' }
		}
```

> Step 2. Add the dependency

```Gradle
implementation 'com.github.MrNouri:RotateView:1.0.0'
```
